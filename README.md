# Sistema Multiagente para Inversión en Mineras BVL

Sistema de soporte a la toma de decisiones basado en Swarm Intelligence con PySwarms.

## Componentes
- 4 Agentes especializados (Técnico, Commodities, Sentimiento, Riesgo)
- PSO Consensus Engine (50 partículas, 100 iteraciones)
- Agente Coordinador

## Tecnologías
- Langflow
- PySwarms (GlobalBestPSO)
- DeepSeek API
- Yahoo Finance (yfinance)
- NewsAPI
- BCRP API
